# MapleStory2-Chinese
此專案將提供中文化的MapleStory 2

使用漢化請將GameOption.xml裡的language index 改為3

徵求懂韓語、中文和想協助維護的人

目前缺人、獨力完成，只能分散更新
## 檔案說明
Lastest-KMS2: 提取原始Xml，最新版本

Translated: 已翻譯的Xml，使用Google Translate和Papago翻譯

## Pull-Request 規則
僅可修改string\cn裡的檔案，其他檔案一率不接受修改
Latest-KMS2一律限專案擁有著修改
